ArcGIS WebApp Builder

Key Features

Create your own intuitive, fast and beautiful web apps without writing a single line of code.

Apps that work anywhere on your desktop, tablet and mobile device.

No plugins required, pure HTML all the way.

The world is not flat; ArcGIS WebApp Builder handles both 2D and 3D.

GIS power tools ready to go with over 30 out-of-the-box widgets.

Extensible framework so you can put your developer skills at work if you choose to.

Allows you to create ArcGIS Web apppcation templates for your ArcGIS Organization.

Works with ArcGIS Online and Portal for ArcGIS.

Mobile Ready

The configured applications run on full desktop browsers, tablets, or mobile devices. ArcGIS WebApp Builder applications detect the dimensions of your display and adjust accordingly. Developers can create their own UI designs for the desktop, tablet, or mobile device.

New 3D Map Support

Configure to enable web 3D capabilities

Supports new 3D JavaScript API (new feature of ArcGIS API for JavaScript)

Supports both plugin and WebGL 3D rendering core

3D enabled widgets are available out-of-the-box

Built-in support for ArcGIS Online and Portal for ArcGIS Organizations

WebMap ready- Applications built with the ArcGIS WebApp Builder leverage your WebMaps. That is, you can author your WebMaps first and build fully functional apps around them.

Register and secure your Apps- Once you have created your application, register it as an item in your ArcGIS Organization, share it with everyone for wide access, or with specific groups if you want to control who can access it.

ArcGIS Web Application Templates Factory- If the ArcGIS WebApplication templates included with ArcGIS Online and Portal for ArcGIS you are not satisfactory, create your own with ArcGIS WebApp Builder. That is right, you can use the over 30+ widgets in ArcGIS WebApp Builder and create your own ArcGIS Web application templates!

HTML5 WYSWYG App Builder

Making a great experience for building your applications is key; therefore, a slick environment has been created for you to quickly put together fully functional web applications. The ArcGIS WebApp Builder is built entirely on HTML, so it can run on any browser. As a configurator you can choose from over 30+ widgets, define the user interface of your tools, and do all of that in a completely WYSIWYG manner. As you add your tools, you can visually see them in your app and use them right away.

Once configuration is completed, you are one click away from having your app running in any device and browser.

Plentiful Capabilities Meeting Industry Needs

Mapping, queries, analysis, Editing, high-quality printing… is included. The ArcGIS WebApp Builder has a nice collection of widgets so you can add in your app what people need. Be careful not to add too much; however, the we richest collection of tools in ArcGIS is available in case they are needed.

Opportunities for Developers

Are you a developer? Then you just found a great framework to boost your productivity building GIS WebApps. You can create your own widgets to extend what your applications can do. In fact, through our Widget Dev Library you can quickly learn how to write custom widgets and embed any functionality you want into your apps. You can also package and share your widgets with others and it will be your decision if you want to do that for free or for a fee.

If that is not enough, you can also create your own application styles so your apps deliver a unique and tailored user interface.
